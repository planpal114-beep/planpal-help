## 자주 묻는 질문

### AI 기능이 작동하지 않아요
설정 > AI 설정에서 Gemini API 키가 올바르게 입력되어 있는지 확인해주세요.
API 키는 [Google AI Studio](https://aistudio.google.com)에서 무료로 발급받을 수 있습니다.

### Apple 캘린더와 동기화가 안 돼요
설정 > 캘린더 연동에서 Apple 캘린더 접근 권한을 허용했는지 확인해주세요.
iPhone 설정 > Planpal > 캘린더 접근을 허용으로 변경해주세요.

### 데이터가 사라졌어요
설정 > 데이터 관리 > 휴지통에서 삭제된 항목을 복구할 수 있습니다.

### Watch 앱이 업데이트되지 않아요
iPhone과 Apple Watch가 같은 Wi-Fi에 연결되어 있는지 확인해주세요.
앱을 완전히 종료 후 다시 실행하면 동기화됩니다.

### 위젯이 표시되지 않아요
홈 화면 길게 누르기 > 위젯 편집에서 Planpal 위젯을 추가해주세요.

## FAQ

### AI feature is not working
Go to Settings > AI Settings and make sure your Gemini API key is correctly entered.
You can get a free API key at [Google AI Studio](https://aistudio.google.com).

### Apple Calendar sync is not working
Go to Settings > Calendar Integration and allow Apple Calendar access.
Or go to iPhone Settings > Planpal > Calendar and set it to Allow.

### My data disappeared
Go to Settings > Data Management > Trash to restore deleted items.

### Watch app is not updating
Make sure iPhone and Apple Watch are on the same Wi-Fi network.
Force-quit and relaunch the app to trigger sync.

### Widget is not showing
Long-press the home screen > Edit Widgets and add the Planpal widget.
